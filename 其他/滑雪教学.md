# 简介

共25步，前10步是入门初学，10-20步是犁式转平行的教学，20-25是高级教学

# 1. 认识雪板

1. 雪板是两头宽中间细，这之间形成的弧度叫自然弧度，自然弧度越大，转弯的自然转弯就越大。回转半径小会更方便转弯。
2. 脚踩的前后有两个固定器，固定器上有释放值，前面的固定器是左右释放的，后面的固定器是前后释放的。释放值应该选择体重(kg)的十位数加减2，比如65kg，就是4-8。初学者建议-1或-2.
3. 这个释放值影响了板子容不容易脱落。越高越越不容易脱落，越低越容易脱落。初学者建议调低，因为摔倒时板子容易脱落。划得比较好了要把这个调高，因为划一些复杂地形的时候，如果调低了，有可能会自动送板，也会发生危险。

# 2. 感受雪板，单脚平地移动

1. 找一块平地，先穿上一只脚，然后试着移动，感受和适应雪板
2. 换另一只脚感受和适应。

# 3. 穿双板

1. 进行平地移动，可以转一圈，也可以前后走一走
2. 重心前后移动，重心很往前，也可以往后，有雪板的支撑，不会倒。
3. 学会感受重心前后移动的时候，板子给我们的反馈
4. 在坡上穿板时，先穿山下脚，也就是靠近坡下的那只脚，可以卡住。

# 4. 平地移动

像滑旱冰一样，外八字地向前走

# 5. 一字登坡

1. 练习上坡，只有会上坡和会刹车了，才可以下坡。
2. 两个板子平行，膝盖向坡上的方向靠，这样雪板的边刃就会卡住雪，并且可以任意一只脚抬起来而不会往下滑
3. 然后就两只腿一只只地抬起来，平行地向上走，当然也可以往山下走。
4. 雪地上会留下一道道的痕迹

# 6. 八字登坡

1. 适用场景：缓一点的破 ，想要爬坡速度更快
2. 就是膝盖内扣，形成一个外八字

# 7. 犁式刹车（八字刹车）

1. 膝盖内扣，内八字，形成的尖角向滚落线的方向
2. 滚落线就是落差最大的那条线，放个小球，自然滚落的方向就是滚落线
3. 角度开的越大，膝盖越往里扣，刹车力量越强
4. 刹车的时候，两个板头不要碰撞，保持一拳距离，否则容易摔倒

# 8. 八字平行控速

1. 在比较缓的坡练习
2. 八字刹车就会停止
3. 减小八字，就减小了板刃刻雪的程度，就会稍微加速
4. 做几次平行换八字，八字换平行，加速减速的过程，直到你可以控制在什么坡度用什么力量
5. 做好这一步，就可以坐初级雪道的缆车了

# 9. 坐缆车

1. 包厢缆车最简单，因为可以摘掉板子
2. 也可以坐座椅拖钎，会比包厢缆车稍微难一点，抓住t把，用它拖住屁股，不用坐下去，然后重心保持在板子中间，手可以抓住t把
3. 座椅缆车，上缆车的时候，两个板子要平行，不要八字了，到站的时候要把雪杖拿高，因为到站的位置经常比较矮，有可能雪杖会磕到地上，把人从缆车上磕下来。
4. 到达山顶以后，通常是一个小下坡，要犁式刹车刹住就可以了。

# 10. 如何正确摔倒

1. 这个非常重要。
2. 摔倒的时候，膝盖是最容易受伤的，要尽量保证摔倒的时候膝盖不去扭动，因为像十字韧带、半月板这些比较严重的伤，都是一个扭的伤，膝盖前后运动没问题，一旦扭了，很容易受伤。
3. 摔倒的时候，不要内八字地坐着摔倒，非常危险。
4. 要侧面倒，往坡上的方向。
5. 发现自己将要失去控制的时候，就主动倒地。

# 11. 犁式转弯（八字转弯）

1. 在犁式直划的基础上，加上左右转弯，就是力量的分配
2. 稍微内八字的时候，力量分配到右腿，右板就会往左指，也就会往左转弯
3. 同理，力量分配到左腿，左板就会往右指，也就会往右转弯
4. 当我们垂直于滚落线的时候，重心抬高，板子就会带着我们转向滚落线的方向向下划。
5. 当我们顺着滚落线的方向向下划的时候，重心降低并向右（双手摸右膝盖），雪板和身体就会向左走。
6. 要点1：力量放在山下脚，重心在右脚，右脚带着我向左转，重心在左脚，左脚带着我向右转
7. 要点2：双手摸山下脚的膝盖，让上身尽量去面朝山下，形成上下身分离
8. 要点3：入弯的时候，重心向下，给板子施压，出弯的时候，重心向上，释放板底的压力。

# 11.1 平行刹车概述

1. 后面大约十个动作都是为了从八字刹车转换到平行刹车
2. 八字刹车膝盖内扣是有一个角度限制的，所以立刃不会太高，刹车的力量就有限。
3. 平行刹车可以立刃很高，刹车的力量就很强
4. 八字刹车的刹车方式伤膝盖，平行刹车不会。
5. 由于以上原因，要划更高的速度，更陡的雪道，就一定要学会平行刹车

# 12.  静止状态大角度立刃

1. 找一个稍微陡一点的坡，站好。
2. 膝盖向坡上内扣，练习立刃，尽量立的高，两个方向都去试一下。
3. 也可以抬起一只脚，分别尝试两只脚单独立刃。
4. 感受一下八字刹车时的两个内刃立起变为平行刹车的两个雪板的同侧刃立起。
